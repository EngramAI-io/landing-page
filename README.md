# EngramAI Landing Page

A high-performance landing page for EngramAI.io built with Next.js 14, featuring an "Apple meets Cyberpunk" design aesthetic.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Components**: Shadcn/UI (Radix Primitives)
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Font**: Geist Sans (Vercel)

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - React components
- `public/` - Static assets (logo, hero demo GIF)
- `lib/` - Utility functions

## Design System

- **Background**: `#050505` (brand-black)
- **Cards**: `#1A1A1A` (brand-gray)
- **Accent**: `#00FF94` (Sentinel Green)

## Notes

- Logo should be placed in `/public/logo.svg`
- Hero demo GIF should be placed in `/public/hero-demo.gif`
